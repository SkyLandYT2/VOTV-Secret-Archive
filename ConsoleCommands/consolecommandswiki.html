Console Commands

Description:

The command console is a console in which you can enter in certain commands to perform different functions and actions. In the top right of the console will give you a list of the common and important commands that you would be using throughout your gameplay however there are secret and hidden commands that perform specific actions. This webpage only lists the hidden and secret commands, not the common ones that you can find easily.

Secret Commands:

Command Description Image
Maxwell The command "maxwell", will spawn a Maxwell Object anywhere in the world except for out of bounds, hard to reach, or glitchy spots. The spawn location will be completely random for each time you type in this command.
Argemwell The command "argemwell", will spawn a Argemwell Object anywhere in the world except for out of bounds, hard to reach, or glitchy spots. The spawn location will be completely random for each time you type in this command.
Alien The command "alien", used to spawn an alien cutout prop right behind you in the older versions, but instead now it spawns the prop behind a any random door that you open. Meaning if you were to spam the command a bunch, you will be able to trigger and spawn the prop behind every door in the world. It will have an output message of "Err; try again"
Execute_duende51 The command "execute_duende51" is no longer in use hence making it unusable, however in previous versions of the game it had the ability to teleport the player into the 51 Map in which the Defenders minigame will start. In the current and newer versions of the game the command now tells you that "Aliens are dead".
Ariral The command "ariral", would spawn an Ariral Entity near where the Ariral Treehouse Landmark would spawn. However, in newer versions of the game, the command would no longer spawn an Ariral entity instead only prompting you that they are "busy""
Ariral.pls The command "ariral.pls", would spawn an Ariral Entity near where the Ariral Treehouse Landmark would spawn. This command essentially replaces the Ariral command in the newer versions.
Llama.saatana The command "llama.saatana", would spawn a Llama plush prop behind you. As soon as you approach the prop the prop will scream, glow red, and jump towards you. Once through you the prop will stop glowing red and return to normalcy. The command can only be ran once otherwise it will give you a message stating that "the llama soul is out there."
Gooseworx.rufus The command "gooseworx.rufus", would spawn an Rufus Entity near the SCP 432 Object. Once it is spawned, creepy music will play. The music that is played is a reference to the FNAF 1 Ambience track.
Gooseworx.thiccfus The command "gooseworx.thiccfus", would spawn an Thiccfus Entity near the SCP 432 Object. Once it is spawned, creepy music will play. The music that is played is a reference to the FNAF 1 Ambience track.
Rufus.fuckoff The command "rufus.fuckoff", will kill any Rufus or Thiccfus Entity that exists in your world and it will stop the creepy music generated from it.
Rep.debug The command will display your current Ariral Rep statistics, including tier and amount.

Hidden Subcommands:

Command Description Image
Sv.target alpha Targets a server at point X: -3.50, Y: -0.50.
Sv.target chair Targets a server at point X: -4.25, Y: -0.50.
Sv.target root Targets a server at point X: -4.25, Y: -0.25.
Sv.target zulu Targets a server at point X: 9987.82, Y: 8021.53.
Sv.target maxwell Targets a server at the Fake Maxwell Landmark.
Tr.reset The command in earlier versions of 0.6.0 used to reset Transformers remotely without having the need for the player to run up and reset it in-person. But ever since 0.6.0a the command no longer has any effect other than being accepted by the terminal.
